рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃБрЃџрЃўрЃљ рЃгрЃўрЃњрЃюрЃерЃў рЃљрЃ«рЃАрЃюрЃўрЃџрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃўрЃЊрЃћрЃюрЃбрЃБрЃарЃљрЃЊ рЃЊрЃљ **рЃљрЃа рЃерЃћрЃўрЃфрЃљрЃЋрЃА Heroku-рЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА**.

## рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃЊрЃљ рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

### ­ЪЊЂ django_project/
Django рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃцрЃЮрЃџрЃЊрЃћрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃћрЃЉрЃА.

- **settings.py** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃћрЃЉрЃА, рЃЏрЃљрЃЌ рЃерЃЮрЃарЃўрЃА рЃЉрЃљрЃќрЃўрЃА, template-рЃћрЃЉрЃўрЃА рЃЊрЃљ рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА.
- **urls.py** РђЊ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃа URL рЃЏрЃљрЃарЃерЃарЃБрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА posts рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА.
- **asgi.py** РђЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ asynchronous рЃАрЃћрЃарЃЋрЃћрЃарЃћрЃЉрЃЌрЃљрЃю рЃАрЃљрЃЏрЃБрЃерЃљрЃЮрЃЊ.
- **wsgi.py** РђЊ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ production рЃњрЃљрЃарЃћрЃЏрЃЮрЃерЃў Django рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃАрЃљрЃерЃЋрЃћрЃЉрЃљрЃЊ.
- **\_\_init\_\_.py** РђЊ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃЏ рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃљрЃарЃўрЃА Python package.

### ­ЪЊЂ posts/
Django рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА (posts) рЃџрЃЮрЃњрЃўрЃЎрЃљрЃќрЃћ.

- **models.py** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА `Post` рЃЏрЃЮрЃЊрЃћрЃџрЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃдрЃгрЃћрЃарЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
- **views.py** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА view-рЃА, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљрЃА template-рЃерЃў.
- **urls.py** РђЊ рЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃА URL рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃА view-рЃћрЃЉрЃЌрЃљрЃю.
- **admin.py** РђЊ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА `Post` рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА Django Admin рЃърЃљрЃюрЃћрЃџрЃўрЃЊрЃљрЃю.
- **tests.py** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃбрЃћрЃАрЃбрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА Post рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃАрЃгрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљрЃА.
- **apps.py** РђЊ posts рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃўрЃА рЃцрЃљрЃўрЃџрЃў.
- **migrations/** РђЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА migration рЃцрЃљрЃўрЃџрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЉрЃљрЃќрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљрЃќрЃћ.
- **\_\_init\_\_.py** РђЊ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃЏ рЃцрЃЮрЃџрЃЊрЃћрЃарЃў рЃљрЃарЃўрЃА Python package.

### ­ЪЊЂ templates/
HTML рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃўрЃА рЃцрЃЮрЃџрЃЊрЃћрЃарЃў.

- **base.html** РђЊ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў template, рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃАрЃ«рЃЋрЃљ HTML рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃЏрЃћрЃЏрЃЎрЃЋрЃўрЃЊрЃарЃћрЃЮрЃЉрЃўрЃЌ рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю.
- **home.html** РђЊ рЃњрЃЋрЃћрЃарЃЊрЃў, рЃАрЃљрЃЊрЃљрЃф рЃюрЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃўрЃљ рЃДрЃЋрЃћрЃџрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ (Post).
